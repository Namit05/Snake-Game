# Snake-Game
Welcome to Snake — the game where you’re constantly hungry and one wrong move ends everything.
You control a snake that moves around the screen eating food to grow longer. Sounds easy… until your own tail becomes your biggest enemy.
The longer you survive, the harder it gets.
---

## Features of the Code

- Smooth snake movement using keyboard controls  
- Food spawning at random positions  
- Snake grows after eating food  
- Collision detection (walls and self)  
- Score tracking system  
- Game over condition  

---

## Tech Used

- Python  
- `turtle` module for graphics  
- `random` module for food placement  

---
## How to Play

- Use arrow keys to control the snake  
- Eat food to grow longer  
- Avoid hitting walls  
- Avoid running into yourself  
Survive as long as you can.
